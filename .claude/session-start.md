# 会话启动配置

> 本项目默认使用智谱 GLM-5 模型

## 默认模型设置

**使用 GLM-5 模型进行所有对话和任务。**

## 模型切换

如需切换模型，请明确指定：
- "使用 GLM-5-Turbo 快速回答"
- "切换到 GLM-5-Plus 深度分析"
- "用 Claude Sonnet 审查代码"

## 可用模型

- **GLM-5** (默认) - 平衡性能和质量
- **GLM-5-Turbo** - 快速响应
- **GLM-5-Plus** - 深度推理
- **Claude Sonnet** - 备用模型

---

**🎯 会话开始时请说: "使用 GLM-5 模型"**
