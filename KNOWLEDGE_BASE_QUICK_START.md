#### 操作步骤

**1. 上传新文档**
- 点击右上角 "上传文档" 按钮
- 填写文档信息:
  - 文档标题 (必填，2-200字符)
  - 文档分类 (必填: 景点介绍/政策说明/路线推荐/常见问题)
  - 文档内容 (必填，10-10000字符)
  - 文件类型 (可选，默认text)
- 点击 "上传" 按钮
- 等待处理完成 (通常<2秒)

**2. 搜索和筛选文档**
- 使用顶部筛选栏:
  - 选择分类进行过滤
  - 输入关键词搜索标题
  - 选择状态筛选 (全部/已完成/处理中)
- 结果实时更新

**3. 查看文档详情**
- 点击文档行的 "查看" 按钮
- 查看完整文档信息和元数据
- 包括: ID、分类、状态、分块数、上传时间

**4. 删除文档**
- 点击文档行的 "删除" 按钮
- 确认删除操作
- 系统自动级联删除相关分块和向量

## 🔧 API接口调用

### 1. 上传文档

```bash
curl -X POST http://localhost:8086/api/knowledge/upload \
  -H "Content-Type: application/json" \
  -d '{
    "title": "新文档标题",
    "category": "ATTRACTION",
    "content": "文档内容...",
    "fileType": "text",
    "userId": 1
  }'
```

### 2. RAG检索

```bash
curl -X POST http://localhost:8086/api/knowledge/search \
  -H "Content-Type: application/json" \
  -d '{
    "query": "故宫开放时间",
    "topK": 3,
    "threshold": 0.3,
    "category": ""
  }'
```

### 3. 获取文档列表

```bash
curl http://localhost:8086/api/knowledge/documents?category=ATTRACTION
```

### 4. AI对话 (带知识库增强)

```bash
curl -X POST http://localhost:8086/api/ai/chat/send \
  -H "Content-Type: application/json" \
  -d '{
    "message": "介绍一下长城"
  }'
```

## 📖 文档分类说明

### ATTRACTION (景点介绍)
包含: 景点历史、特色、游览攻略等信息

**示例**: "故宫详细指南"、"长城八达岭段介绍"、"颐和园皇家园林"

### POLICY (政策说明)
包含: 门票政策、退款规定、优惠信息等

**示例**: "退改签政策"、"学生优惠说明"、"购票须知"

### ROUTE (路线推荐)
包含: 一日游、多日游、主题游等路线规划

**示例**: "北京一日文化游"、"亲子两日游"、"深度文化游路线"

### FAQ (常见问题)
包含: 开放时间、交通指南、最佳游览时间等

**示例**: "景点开放时间查询"、"交通指南"、"最佳游览季节"

## ⚠️ 常见问题

### Q1: 文档上传失败怎么办？
**A**: 检查以下几点:
- 文档标题是否为空
- 内容长度是否在10-10000字符之间
- 网络连接是否正常
- 后端服务是否运行

### Q2: 检索结果不准确？
**A**: 可以尝试:
- 降低相似度阈值 (threshold)
- 增加返回结果数量 (topK)
- 使用更具体的查询关键词
- 检查知识库是否包含相关内容

### Q3: AI回复没有引用知识库？
**A**: 可能原因:
- 检索相似度不足，未达到阈值
- 知识库中没有相关内容
- AI判断直接回答更合适

### Q4: 系统响应慢？
**A**: 检查:
- 后端服务负载情况
- 数据库连接是否正常
- 网络延迟
- 文档大小和数量

## 🔒 安全注意事项

1. **不要上传敏感信息**: 避免包含个人隐私、密码等敏感内容
2. **定期备份知识库**: 导出重要的文档数据
3. **权限控制**: 确保只有授权用户可以管理知识库
4. **监控使用情况**: 定期检查系统访问日志
