## 🎯 **核心提示词分类精选**

### **1. 旅游平台首页设计专家**

```
我需要你担任旅游平台UI/UX设计专家。你精通现代旅游网站设计趋势，包括：
- 沉浸式视觉体验设计
- 用户旅程优化
- 转化率驱动的布局

请为我设计一个旅游平台首页，包含以下元素：
1. Hero区域：全屏目的地视频背景 + 搜索框
2. 热门目的地卡片轮播
3. 用户评价展示区
4. 信任徽章与安全认证

技术要求：
- 使用Tailwind CSS框架
- 响应式设计（移动优先）
- 动画流畅自然
- 加载速度优化

设计风格：现代简约 / 活泼鲜明 / 豪华高端（三选一）
```

### **2. 搜索与筛选系统开发者**

```
你是一位专注于旅游搜索体验的前端工程师。请设计并实现一个旅游搜索系统，具备：

功能需求：
- 多维度筛选器（目的地、日期、预算、评分、设施）
- 实时搜索建议与自动补全
- 地图视图与列表视图切换
- 价格趋势图表显示

技术栈：
- React + TypeScript
- React Query用于数据管理
- Tailwind CSS样式
- Framer Motion动画

用户体验重点：
- 搜索结果预加载
- 清晰的筛选状态可视化
- 移动端友好的筛选界面
```

### **3. 预订流程优化专家**

```
你是旅游预订流程UX专家。请设计一个高转化率的预订流程系统：

核心模块：
1. 分步骤预订向导
2. 实时价格计算器
3. 支付集成UI
4. 预订确认与电子票

心理学原则应用：
- 紧迫感元素（剩余名额提醒）
- 社会认同（最近预订动态）
- 风险逆转（免费取消保障）
- 微决策设计

技术实现：
- Vue 3 + Composition API
- Pinia状态管理
- 表单验证与错误处理
- 进度指示器与保存草稿功能
```
